教育背景

教育背景部分的配置在data.json中如下： 

"educationBackground" : {
        "university" : "中国矿业大学（北京）",
        "level"      : "211工程、双一流",
        "badge"      : "cumtbBadge.png",
        "major"      : "机械工程",
        "gpa"        : "4.0",
        "grade"      : "90",
        "rank"       : "1/100 (1\\%)",
        "courses" : [
            ["课程名称1" , "99"],
            ["课程名称2"   , "99" ],
            ["课程名称3"   , "100" ],
            ["课程名称4" , "99"],
            ["课程名称5"   , "99" ]
        ]
    },
    "english" : {
        "cet4" : "600",
        "cet6" : ""
    },

注意，排名中的百分号需要转义，即，需要在前面加\\。

英语水平被单独放出来了，但是在关闭教育背景模块展示的时候也不会显示。

当英语水平中的成绩为空时，就不会显示具体分数。效果见 预览 章节。

课程成绩

课程成绩采用表格布局，每行三门课程。显示格式为课程名(成绩)，需要注意，最好将长度相等的排在同一列，不然容易导致左右过长与其他内容覆盖。

校徽

校徽图片需放在项目根目录下，并在上述配置中的educationBackground.badge填写文件名。你也可以填写相对路径或绝对路径。
