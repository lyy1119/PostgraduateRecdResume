✨ 功能与特点 Features

  • 📄 高质量排版：使用 LaTeX 编译，确保简历专业美观、格式统一
  • 🔄 数据驱动：采用 Lua 脚本，从 JSON 文件加载个人信息，结构清晰，方便快速修改和复用
  • ⚙️ 多种编译方式：支持使用 lualatex、make 命令或基于 Docker 一键编译，省心省力
  • 多种内容板块可选展示
  • 🚧 更多功能开发中
